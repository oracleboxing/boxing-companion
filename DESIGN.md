# Design Direction

## Feel

Premium, minimal, calm, coach-led.

Think Apple Fitness clarity, not noisy fitness SaaS.

## Main interaction

The app should make one action obvious:

**Start Workout**

Everything else is secondary.

## Visual principles

- clear hierarchy
- large readable timer during workouts
- restrained colour
- generous spacing
- simple cards
- no cluttered dashboards
- no RPG/gamification overload

## Motion

Use motion to guide, not entertain.

Good motion:
- smooth timer transitions
- subtle block changes
- calm completion state
- haptics/audio cues later

Bad motion:
- confetti everywhere
- bouncing badges
- frantic streak UI

## Demo/animation future

Future exercise/move demos may use a simple “action man” style loop.

For now, design the UI so each movement/exercise can eventually have:
- title
- short instruction
- coaching cues
- common mistakes
- simple demo visual/animation

Do not block the MVP on animation quality.
